imagen.analysis¶

Module¶

The ImaGen analysis module provides common analysis functions, which can be applied to any SheetView or SheetStack. This allows the user to perform analyses on their input patterns or any other arrays embedded within a SheetView and display the output of the analysis alongside the original patterns.

Currently this module provides FFT, auto-correlation and gradient analyses.

class imagen.analysis.contours(**params)[source]¶

Bases: dataviews.operation.ViewOperation

Given a SheetView with a single channel, annotate it with contour lines for a given set of contour levels.

The return is a overlay with a Contours layer for each given level, overlaid on top of the input SheetView.

param NumericTuple levels (constant=False, default=(0.5,), instantiate=False, length=1, pickle_default_value=True, precedence=None, readonly=False)
A list of scalar values used to specify the contour levels.
class imagen.analysis.cyclic_similarity_index(**params)[source]¶

Bases: dataviews.operation.ViewOperation

The similarity index between any two cyclic maps. By default, a zero value indicates uncorrelated SheetView data. The similarity index may be useful for quantifying the stability of some cyclic quantity over time by comparing each sample in a SheetStack to the final SheetView element.

param Boolean unit_range (allow_None=False, bounds=(0, 1), constant=False, default=True, instantiate=False, pickle_default_value=True, precedence=None, readonly=False)
Whether to scale the similarity values linearly so that uncorrelated values have a value of zero and exactly matching elements are indicated with a value of 1.0. Negative values are then used to indicate anticorrelation.
