topo.tests.unit.testplot¶

Module¶

Test for the Plot class.

topo.tests.unit.testplot.matrix_hsv_to_rgb(hMapArray, sMapArray, vMapArray)[source]¶

First matrix sets the Hue (Color). Second marix sets the Sauration (How much color) Third matrix sets the Value (How bright the pixel will be)

The three input matrices should all be the same size, and have been normalized to 1. There should be no side-effects on the original input matrices.
