class topo.tests.unit.testcfsom.ImageSaver(**params)[source]¶

Bases: topo.base.simulation.EventProcessor

A Sheet that receives activity matrices and saves them as bitmaps. Each time an ImageSaver sheet receives an input event on any input port, it saves it to a file. The file name is determined by:

<file_prefix><name>_<port>_<time>.<file_format>

Where <name> is the name of the ImageSaver object, <port> is the name of the input port used, and <time> is the current simulation time.

Parameters::
file_prefix = (default ‘’) A path or other prefix for the
filename.
file_format = (default ‘ppm’) The file type to use when saving
the image. (can be any image format understood by PIL)

time_format = (default ‘%f’) The format string for the time. pixel_scale = (default 255) The amount to scale the

activity. Used as parameter to PIL’s Image.putdata().
pixel_offset = (default 0) The zero-offset for each pixel. Used as
parameter to PIL’s Image.putdata()
