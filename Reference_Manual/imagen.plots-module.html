class imagen.plots.Plot(**kwargs)[source]¶

Bases: param.parameterized.Parameterized

A Plot object returns either a matplotlib figure object (when called or indexed) or a matplotlib animation object as appropriate. Plots take view objects such as SheetViews, SheetLines or SheetPoints as inputs and plots them in the appropriate format. As views may vary over time, all plots support animation via the anim() method.

class imagen.plots.ProjectionGridPlot(grid, **kwargs)[source]¶

Bases: imagen.plots.Plot

ProjectionGridPlot evenly spaces out plots of individual projections on a grid, even when they differ in size. The projections can be situated or an ROI can be applied to each element. Since this class uses a single axis to generate all the individual plots it is much faster than the equivalent using subplots.
